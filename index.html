✍️ How to Use

1. Start Writing

Just type normally — no formatting needed.

2. Scene Heading

INT. ROOM - DAY

3. Character

JOHN

4. Dialogue

Hello, how are you?

5. Action

John walks into the room.

Basic Rules

• Use CAPS for characters

• Start scenes with INT / EXT

• Dialogue comes after character

• Press Enter to continue writing

Templates

Use “Screenplay Template” to start structure.

⚠ Important:

Export your script as .fountain before changing template.
Template change will overwrite your current script.

Preview

Switch to Preview tab to see formatted script.

Saving Your Work

⚠ No Auto Save:

This tool does NOT auto save your script.

Always export your script as .fountain file to save your work.

To continue later, use Open .fountain to reload your script.

Tip: If it looks like a screenplay, you're doing it right.
